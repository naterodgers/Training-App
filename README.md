# Day Won

AI-powered endurance training app — from your first 5K to a Full Ironman.

## Setup

1. Open `src/App.jsx` and replace `YOUR_API_KEY_HERE` with your Anthropic API key
2. Run `npm install`
3. Run `npm run dev` to start locally

## Deploy to Vercel

Push to GitHub, then import the repo at vercel.com. No config needed — Vite is auto-detected.
